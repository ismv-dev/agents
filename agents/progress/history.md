# Historical Log (append-only)

> Each time a session ends, its summary is added here.
> Do not edit previous entries. Only add to the end.
> Expected structure:
```
## <YYYY-MM-DD> — <Name of the increment/feature>
- **Agent:** <Agent Name>
- **Plan:** <Implementation Plan>
- **Changes:** <Changes made>
- **Status:** <Status of the increment/feature>
- **Result:** <New feature(s) of the increment/feature or details of the current state>
- **Verification:** <Validation criteria applied and their results>
```
