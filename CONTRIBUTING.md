# Contributing

Gracias por querer contribuir. Por favor, sigue estos pasos:

1. Abre un issue describiendo el cambio o mejora.
2. Crea una rama con nombre `fix/<breve-descripción>` o `feat/<breve-descripción>`.
3. Añade o edita archivos respetando la plantilla de agentes en `.claude/agents/template.md`.
4. Abre un Pull Request con descripción clara y cambios agrupados.

Normas rápidas:

- Mantén descripciones claras en frontmatter (`name`, `description`, `tools`, `model`).
- Incluye ejemplos de input/output cuando aplique.

Plantilla de PR mínima:

- Resumen corto
- Qué problema resuelve
- Cambios realizados
- Cómo probar
